Global Linking

Lets start linking!

There are two ways to connect your Bedrock account to your Java account.

Online account linking

The first way to link your account is by logging in to your Bedrock and Java account online, which verifies that you have access to both accounts.

Joining our global linking server

By logging in to the global linking server with both your Bedrock and your Java account we know that you have access to both accounts.

Both linking methods should take less than five minutes.
